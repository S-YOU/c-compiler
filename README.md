# c-compiler
seccamp2018 c compiler

[日記](https://github.com/hsjoihs/c-compiler/blob/master/diary.md)

## Environment

Tested on both macOS (High Sierra) and Ubuntu (18.04.1 LTS).

## How to run

After cloning, run:

```
$ make init
$ make notest
```

to compile.

## How to test

Run

```
$ make test_all_
```
